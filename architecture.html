Folder Structure

How the codebase is organized for scalability and maintainability. 

intelli-credit-ai/
├── backend/                  # FastAPI Backend Service
│   ├── app/
│   │   ├── main.py           # API Entry Point
│   │   ├── api/              # API Routes (upload, analysis, etc.)
│   │   ├── core/             # Config, Security, DB Connections
│   │   ├── services/         # Business Logic layer
│   │   └── tasks/            # Celery Background Workers
│   ├── requirements.txt
│   └── Dockerfile
│
├── frontend/                 # Next.js Dashboard UI
│   ├── app/                  # Pages and Routes
│   ├── components/           # Reusable React Components
│   ├── lib/                  # API Clients and Utilities
│   ├── public/               # Static Assets
│   ├── package.json
│   └── Dockerfile
│
├── ml/                       # Machine Learning Models
│   ├── models/               # XGBoost, FinBERT weights
│   ├── pipelines/            # Data Prep & Processing Scripts
│   ├── agents/               # LangChain Research Agents
│   └── notebooks/            # Jupyter Training Notebooks
│
├── webpage/                  # Showcase Landing Page (HTML/CSS)
│   ├── index.html            # Main Showcase
│   ├── architecture.html     # Structure Deep Dive
│   └── style.css             # Global Styling
│
├── uploads/                  # Temporary Document Storage
├── docker-compose.yml        # Orchestrates all services
├── .env                      # Environment Variables
└── README.md                 # Project Documentation
